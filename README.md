# Discord AXL Tool

Free open-source Discord tool.

## Features
- Check Tokens
- Logs
- Tool Configurations
- More features (coming soon...)

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/axlxz/discord-axl-tool.git
    ```

2. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Run Script

```bash
python main.py
